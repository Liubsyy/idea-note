## 收纳规则

- 未来要搜索：保留稳定关键词，比如小区名、商户名、证件名。
- 需要定期检查：写清楚日期和提醒节点。
- 带个人偏好的内容：记录“喜欢/不喜欢”和原因，不只写结论。
