# 旅行打包清单

`#备忘` `#旅行` `#清单`

## 出门前一天

- [ ] 身份证、护照或通行证放进随身包
- [ ] 充电宝充满，数据线放透明收纳袋
- [ ] 关闭阳台窗户
- [ ] 清掉冰箱里容易坏的叶菜和熟食
- [ ] 设置扫地机器人避开行李箱区域

## 随身包

| 物品 | 数量 | 备注 |
| --- | ---: | --- |
| 身份证 | 1 | 放内袋，不和钱包分开 |
| 耳机 | 1 | 提前下载播客 |
| 纸巾 | 2 包 | 一包放外侧口袋 |
| 湿巾 | 1 包 | 小包装够用 |
| 常用药 | 1 盒 | 晕车药、肠胃药、创可贴 |

## 行李箱

| 天数 | 衣物建议 | 额外提醒 |
| ---: | --- | --- |
| 1-2 天 | 上衣 2 件、内衣袜子 2 套 | 不带备用鞋 |
| 3-4 天 | 上衣 4 件、裤子 2 条 | 带折叠衣架 |
| 5 天以上 | 按 4 天量循环 | 预留洗衣时间 |

## 回来后

- [ ] 把票据照片放到记账目录
- [ ] 充电宝和转换头放回抽屉
- [ ] 记录这次没用上的东西，下次少带
- [ ] 把好吃的店名记到城市备忘

```text
下次轻装目标：随身包不超过 3kg，行李箱留出 1/4 空位。
```
