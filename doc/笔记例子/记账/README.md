# 记账目录

记录收入、支出、订阅、报销和预算复盘。

## 口径

- 固定支出：房租、水电、保险、长期订阅。
- 可变支出：餐饮、交通、购物、临时服务。
- 报销：先按支出记，到账后在收入中冲抵。
