# my-notes

这是给 **Idea Note** 准备的一组示例笔记，适合用来测试：

- 笔记模式的树形目录与卡片摘要
- 大纲跳转、全局搜索、标签页编辑
- 任务列表、表格、Mermaid、KaTeX、HTML、代码块、引用与相对链接
- 计划、备忘、记账三类日常内容的混合工作流

## 快速入口

| 分类 | 入口 | 适合测试 |
| --- | --- | --- |
| 计划 | [季度目标](./计划/2026-Q3-季度目标.md) | 大纲、任务、甘特图、公式 |
| 计划 | [产品迭代排期](./计划/Idea-Note-产品迭代排期.md) | Mermaid、表格、代码块 |
| 备忘 | [周末采购备忘](./备忘/周末采购备忘.md) | 表格、Mermaid、任务 |
| 备忘 | [生活缴费与证件到期](./备忘/生活缴费与证件到期.md) | 表格、提醒、流程图 |
| 记账 | [家庭记账台账](./记账/2026-07-家庭记账.md) | 表格、公式、分类汇总 |
| 记账 | [订阅与软件成本](./记账/订阅与软件成本.md) | 预算、表格、提醒 |
